Experiencia con los repositorios

GITHUB

  • Fue el primer repositorio que use y he estado usando durante mucho tiempo.
  • El manejo es sencillo una vez uno se aconstumbra.
  • Cuenta con una aplicacion para la mayoria de sistemas que simplifica muchisimo el trabajo.
  • La subida es rapida

BITBUCKET

  • Tiene una interfaz muy intuitiva y amigable que difiere de los demas.
  • El manejo dada su interfaz es mucho mas sencillo.
  • La subida es rapida

GITLAB

  • La subida es la misma que los otros repositorios con el agregado que presenta el comando para cambiar el origen
  • Tiene muchos mas niveles de seguridad que son permisivos al hacer algunas cosas pero supongo que si uno se adapta y necesita esto puede ser una opcion a tener cuenta.
  • La subida es rapida
